[IconX] NormallyOff

[IconX] NormallyOn

NormallyOff or NormallyOn (only one should be used per icon) can be used to reverse the normal action of an icon. By default, Icons are normally Off, and are set to On when they are activated. By setting NormallyOff=False or NormallyOn=True, you can reverse this situation, causing an icon to turn Off when activated.

Default value: True for NormallyOff, False for NormallyOn.
