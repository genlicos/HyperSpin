Hatari for Windows was built using the mingw32 version of the gcc compiler,
using the following libraries (those dll were also built with mingw32-gcc) :

SDL.dll			version 1.2.14 - October 20, 2009
http://www.libsdl.org/download-1.2.php

libpng-3.dll		version 1.2.35 - February 14, 2009
http://sourceforge.net/projects/libpng/

readline.dll		version 5.2 - October 11, 2006
http://www.gnu.org/s/readline/

libtermcap-0.dll	version 1.3.1 - March 3, 2002
http://ftp.gnu.org/gnu/termcap/

zlib1.dll		version 1.2.3 - July 18, 2005
http://zlib.net/
